# springboot-okta-example
- Okta와 Springboot를 연동한 인증 예제

## Tech
- Springboot 2.6.3
- Spring Security 5.6.1
- Okta Spring Boot Starter 2.1.4


## Guide
1. Okta 애플리케이션 등록
2. application.yml에 client-id, client-secret, okta domain등록(okta 도메인 형식 : https://${mydomain})
